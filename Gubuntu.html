Setup Grafana on Ubuntu OS

We install on Ubuntu22

Lets Start by


  1. We open the terminal and type
    sudo apt update && sudo apt upgrade -y
    to refresh our Update the package list to ensure you have the latest version of all packages

  2. To install all the Dependencies for grafana we type on the terminal
    sudo apt install -y apt-transport-https software-properties-common

  3. We type the command to Add the Grafana APT repository and import its GPG key.
    wget -q -O - https://packages.grafana.com/gpg.key | sudo apt-key add - echo "deb https://packages.grafana.com/oss/deb stable main" | sudo tee /etc/apt/sources.list.d/grafana.list

  4. Update the package list again and install Grafana.
    sudo apt update sudo apt install grafana -y

  5. Start the Grafana service and enable it to start on boot.
    sudo systemctl start grafana-server sudo systemctl enable grafana-server

  6. Check the Grafana service status to ensure it is running:
    sudo systemctl status grafana-server

  7. Open your web browser and navigate to http://:3000.
    Log in with the default credentials:
    Username: admin
    Password: admin (you'll be prompted to change it after the first login).
